Présentation des concepts

Qu'est-ce qu'un développeur web ?

Un développeur web est une personne chargée de réaliser des interfaces qui vont fonctionner sur le web. Ces interfaces peuvent être visibles, comme elles peuvent être invisibles.


Le développeur qui réalise des interfaces visibles est appelé développeur frontend. Tandis que l'autre est appelé développeur backend.


Toutefois, il y a également un troisième type de développeur web appelé développeur full stack, dont la mission est de créer les deux types d'interfaces.


Qu'est-ce qu'un graphic designer ?

Un graphic designer est une personne dont la mission est de combiner et d’organiser les éléments visuels pour créer un message efficace et attrayant.


Qu'est-ce qu'un beatmaker ?

Un beatmaker est une personne qui combine plusieurs mélodies pour creer un son fini appelé instrumentale (ou beat en anglais).
